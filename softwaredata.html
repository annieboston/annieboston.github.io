Tissues, colorectal cancer, and mouse CHARM data

Data is described in our CpG shore paper [PMID: 19151715]. These are from arrays described in our CHARM paper [PMID: 18316654].

  • XYS files for human samples [gzipped tar archive, 1.7 GB]
  • sample information for human samples [csv file]
  • XYS files for mouse samples [gzipped tar archive, 296 MB]
  • sample information for mouse samples [tab delimited file]
  • CHARM ndf file
  • CHARM pos file
  • Mouse CHARM ndf file
  • Mouse CHARM pos file
  • Illumina data used in CHARM paper
  • Charm paper DKO and HCT116 data [gzipped archive, 235 MB]
  • Example for R package

 

Spike-ins from Affymetrix, Agilent, and Illumina

Data is described in our platform comparison using spike-ins paper. Nucleic Acids Research 2008 [PMID: 18676452]

  • Affymetrix [gzipped tar archive, 139 MB]
  • Agilent [gzipped tar archive, 161 MB]
  • Illumina [gzipped tar archive, 18 MB]

 

Affymetrix SNP chip data

The Data is available from the SNPaffycomp site.
