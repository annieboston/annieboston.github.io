Software

Bump hunting

Code described in paper Bump hunting to identify differentially methylated regions in epigenetic epidemiology studies is available from the Bioconductor bumphunter package. The key function is bumphunterEngine

fRMA 

frozen RMA (frma): is an R packge for preprocessing and quality control of affymetrix probe-level data for hgu133a arrays that can process arrays individually.

CHARM

 

R package for CHARM

crlmm for Illumina

 

The chip-specific packages used by the crlmm package to process data from the following Illumina Infinium BeadChips are available from the Bioconductor project.

  • 550k (human550v3bCrlmm)
  • 650k (human650v3aCrlmm)
  • 1M (human1mv1cCrlmm)
  • 370k Duo (human370v1cCrlmm)
  • 1M Duo (human1mduov3bCrlmm)
  • 370k Quad (human370quadv3cCrlmm)
  • 610k Quad (human610quadv1bCrlmm)
  • 660k Quad (human660quadv1aCrlmm)
  • Omni1 Quad (humanomni1quadv1bCrlmm)

 

A data package containing example idat files from 40 HapMap samples named hapmap370k is also available from Bioconductor.

 

Bioconductor packages

 

Most software is available from Bioconductor. Bioconductor is an open source and open development software project for the analysis and comprehension of genomic data.

 

Example of packages related to the software described in this page are:

  • affy: preprocessing methods for Affymetrix expression arrays, e.g. RMA.
  • gcrma: sequence based background correction for Affymetrix expression arrays.
  • oligo: preprocessing methods for Affymetrix and Nimblegen arrays.
  • crlmm: genotype calling and copy number analysis for Affymetrix and Illumina SNP arrays.
  • spkTools: assessment methods for spike-in experiment data.
  • affycomp: engine software for affycomp website.

All are available from the Packages link at http://www.bioconductor.org
